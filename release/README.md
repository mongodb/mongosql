# Requirements

- Evergreen username and key (available at https://evergreen.mongodb.com/settings)
- AWS API access key and secret (you need permissions on the `info-mongodb-com` S3 bucket)

# Process

Run `release.py` and follow the prompt to release a new version of the BI Connector.

Verify the information on the downloads site at ```https://www.mongodb.com/download-center#bi-connector``` is correct and that all URLs are working correctly.
